# AnalogMovementVS
This mod adds analog movement to Vintage Story so you can move in any direction.
This must be installed on the server and client to work.
This does nothing on its own and needs a child mod.


This mod overwrites SystemPlayerControl.OnGameTick() and replaces EntityControls for EntityPlayer.

[help with making a child mod](https://github.com/ZechMB/AnalogMovementVS/blob/main/extra/HowMakeChild.txt)
